pyFRB/US Documentation¶

pyFRB/US is a Python-based simulation platform for the Federal Reserve Board’s FRB/US model.

User guide¶

The user guide provides information on routine usage of the pyFRB/US package, including example code and in-depth information on simulation options.

pyFRB/US User Guide

FRB/US Model Object API¶

The API documentation for the FRB/US model object gives detailed descriptions of methods, arguments and usage of the Frbus object.

Frbus Object API

Time-Series Operation Documentation¶

This page describes the TimeSeriesData object and how it is used to execute recursive time-series operations on data via a declarative syntax.

Time-Series Operation Documentation

Contact¶
